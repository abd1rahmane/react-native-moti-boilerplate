## Prerequisites

- Node.js >= 14
- yarn
- Expo CLI
- iOS Simulator or Android Emulator

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/react-native-moti-boilerplate.git
```

2. Install dependencies:
```bash
yarn install
```

3. Start the development server:
```bash
yarn start
```
